Etherscape

Etherscape is a behavioral system that interacts with wireless networks into a perceptible and sensory encounter by converting data into audio signals. This allows the user to encounter the intangible networks in urban surround- ings and adds an additional layer to the current physical environment. Using this device, users can explore urban spaces by sensing the strength and variability of wireless signals.

Fragmentation

The fragmentation tool is aimed for graphic designers wanting to bring a bit more dadaism into their designs. Graphic designers are known to be pixel perfect and striving for perfectly kerned typography, this tool will bring some randomness and a good amount of chaos. The main way to interact with the tool and apply distortion will be a rectangular drag selections tool which the user controls with their mouse. The tool will be accompanied with a graphical user interface reminiscing of graphic design tools such as photoshop, illustrator etc. to give the user a sense of familiarity. The main futures of the tool will be displacement, pixelation, and blur. The blur and pixelate tool will give more layered feeling to the overall design by adding more textures.

Daiku

Daiku explores the intersection of Dadaist and Haiku poetry through generative methods. Inspired by Hugo Ball’s sound poetry and Tristan Tzara’s cut-up technique, Daiku assembles haikus by selecting phrases from random Wikipedia entries, using syllable-count analysis to ensure adherence to Haiku’s 5-7-5 format. The project incorporates consonant-vowel patterns to refine phonetic consistency and visual coherence across lines. Each generated haiku is paired with abstract visuals created using a Stable Diffusion model, balancing randomness with structured presentation.
